##### Height vs Nodes of m-ary Tree
**Height given**
min no. of nodes : n = mh+1
max no. of nodes : n = $(m^{h+1}-1)/(m-1)$

**Nodes given**
min height : h = $\log_{m}{[n(m-1)+1]}-1$
max height : h = $(n-1)/m$
